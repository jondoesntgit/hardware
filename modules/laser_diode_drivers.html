Laser Diode Drivers¶

This module provides support for controlling laser diode drivers using the Python language.

Laser diode drivers can be imported by

>>> import hardware
>>> ldd = hardware.laser_diode_drivers.ILX_Lightwave_3724B('GPIB0:...')
class hardware.laser_diode_drivers.ILX_Lightwave_3724B(visa_search_term)[source]¶

Hardware wrapper for ILX Lightwave 3724 laser diode driver

Parameters:visa_search_term (str) – The address that is passed to visa.ResourceManager().open_resource()
Variables:current (float) – The current of the laser diode driver
