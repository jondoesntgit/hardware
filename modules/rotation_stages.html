Rotation Stages¶

This module provides support for controlling rotation stages with Python. A rotation stage can be imported by

>>> import hardware
>>> rot = hardware.rotation_stages.NSC_A1(port='COM3')
class hardware.rotation_stages.NSC_A1(port=’COM3’, channel=1, timeout=0.1, max_x=300000.0, min_x=-300000.0, verbose=False)[source]¶

Hardware wrapper for NSC-A1 Single Axis USB Stepper Motor Controller

The stepper motor is a USB to Serial interface. On Windows, you communicate with it over a COM port, where on Unix, you would communicate over a /dev/tt port.

Parameters:
  • port (str) – The serial port over which to communicate with the device.
  • channel (int) – Multiple axes can be controlled by the motor
  • Which channel is this rotational stage talking on? (controller.) –
  • timeout (float) – How many seconds should the serial interface wait for a response before timing out?
  • max_x (int, float) – The maximum x (in ticks) to rotate to. There are 10,000 ticks in one degree. Rotations beyond this bound will not occur and will raise exceptions
  • min_x (int, float) – The minimum x (in ticks) to rotate to. There are 10,000 ticks in one degree.
  • verbose (bool) – If verbose is True, then a progress bar will show while the stage is rotating.
Variables:
  • x (int) – The absolute position of the rotation stage in ticks
  • angle (float) – The absolute positon of the rotation stage in degrees
  • max_x (int) – The maximum position the stage can rotate to without raising an exception.
  • min_x (int) – The minimum posiiton the stage can rotate to without raising an exception
  • max_angle (float) – The maximum angle the stage can rotate to without raising an exception
  • min_angle (float) – The minimum angle the stage can rotate to without raising an exception
  • velocity (float) – The velocity of the stage in degrees per second.
